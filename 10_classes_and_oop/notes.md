# Classes and JavaScript


## Object-Oriented Programming (OOP) 

It is a programming paradigm that uses "objects" to design applications and computer programs. It utilizes several key concepts, including:
- **Classes**: A blueprint for creating objects. It defines a set of properties and methods that the created objects will have.
- **Objects**: Instances of classes. They can hold data and have behavior defined by the class.
- **Abstraction**: The concept of hiding the complex implementation details and showing only the necessary features of an object. It allows the programmer to focus on interactions at a higher level without needing to understand the intricate workings of the object.
- **Encapsulation**: The bundling of data and methods that operate on that data within one unit, e.g., a class. It restricts direct access to some of the object's components, which can prevent the accidental modification of data.
- **Inheritance**: A mechanism where one class can inherit properties and methods from another class. This promotes code reusability and establishes a relationship between classes.
- **Polymorphism**: The ability of different classes to be treated as instances of the same class through a common interface. It allows methods to do different things based on the object it is acting upon.

## Why Use OOP?
- **Modularity**: The source code for an object can be written and maintained independently of the source code for other objects. Once created, an object can be easily passed around inside the system.
- **Reusability**: Objects can be reused across programs. The use of inheritance and polymorphism allows for the creation of new classes based on existing ones, promoting code reuse.
- **Pluggability and debugging ease**: If a particular object turns out to be problematic, you can simply remove it from your application and plug in a different object as its replacement. This makes it easier to debug and maintain the code.

## parts of OOP
- **Class**: A class is a blueprint for creating objects. It defines the properties and methods that the objects created from the class will have. In JavaScript, classes can be defined using the `class` keyword, and they can include a constructor method for initializing new objects.
- **Object**: An object is an instance of a class. It is created using the `new` keyword followed by the class name. Objects can have their own properties and methods, which are defined by the class they are instantiated from.
